# Importing Moby Thesaurus

1. Download it: http://www.gutenberg.org/ebooks/3202
2. Import it: `$ rake import[mthesaur.txt]` (_assuming the unzipped thesaurus file is sitting in your rails root named `mthesaur.txt`_)
